## Hardware Overview Section

### Headline
**Industrial Hardware Built for RES Plants**

### Subheadline
No vendor lock-in. No commercial protocol limitations. Just pure interoperability.

### Core Description
At the core of +SolarControl is an industrial MiniPC with custom firmware, housed in a sealed, pre-wired panel built to EN IEC 61439-1 & EN 61439-5 standards.

It installs in minutes, operates in extreme temperatures (-20 to +60°C) and securely manages all communication with field devices.

### Key Hardware Features
- Custom Firmware — OTA updates & flexible configuration
- Edge Controller — unique identifier & VPN client for secure access
- Zero-Trust Security — impervious to cyberattacks & tampering attempts
- Industrial Durability — certified, sealed, mechanically protected
- Compatibility with 65+ inverter manufacturers
- Support for all major open protocols (IEC-60870-5-104, Modbus TCP/RTU, IEC61850, OPC-UA/DA/XML-DA)
- Sealed enclosure for physical protection
- Integration with aggregators
- Ready for storage/BESS control
- API for future interoperability

### Visual Requirements
- Installation photos of hardware in real-world settings
- Clean, technical photography
- Hardware close-up details
