# Case Studies Page Specification - Optimems

A clean, elegant page showcasing Optimems' real-world impact through client success stories. The page uses a simple, refined layout that lets the work speak for itself while reinforcing the company's mission to democratize and accelerate the energy transition.

## Visual Design Guidelines

### Overall Aesthetic
- Clean, editorial design
- Generous whitespace
- Typography-forward
- Minimal decorative elements
- Let the content breathe

### Card Design
- Consistent structure across all cards
- Client name as primary focus
- Brief description (2-3 sentences max)
- Industry tags for quick scanning
- Subtle hover interaction
- Arrow or indicator suggesting more detail available

### Color Palette
- Light, warm background
- Dark text for readability
- Accent color for tags and interactive elements
- Dark section for CTA contrast

### Typography
- Serif font for headlines and client names (elegant, trustworthy)
- Sans-serif for body text and tags (clean, modern)
- Clear hierarchy: Label → Headline → Description → Tags

---

## Responsive Behavior

### Desktop (900px+)
- 2-column grid
- Full navigation
- Spacious padding

### Tablet (640px - 899px)
- 2-column grid or single column
- Condensed navigation
- Adjusted spacing

### Mobile (< 640px)
- Single column layout
- Hamburger or minimal navigation
- Touch-friendly card sizing
- Stacked CTA elements
